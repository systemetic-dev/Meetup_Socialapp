Meetup backend

React frontend + Django API scaffolded.

This Django server exposes meeting data at /api/highlights/ and /api/meetings/. The React application source lives in frontend/ and is designed to consume these endpoints.

Backend

REST endpoints, seed data, and optional MongoDB persistence via pymongo.

Frontend

React + Vite source with a Zoom-style SaaS landing page and meeting dashboard.

Next step

Install frontend dependencies, set MONGODB_URI, then run both servers.
